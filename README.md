## 快速开始

```bash
# 1. 安装依赖
uv pip install -r requirements.txt

# 2. 配置环境变量
cp .env.example .env
# 编辑 .env 填写 Azure OpenAI 配置

# 3. 运行
streamlit run app.py
```
